# UIDAI Biometric Update Demand Prediction

This project analyzes Aadhaar Biometric Update data to identify regional and temporal patterns in biometric update demand.

## Objective
To predict biometric update demand using data analytics and machine learning and support proactive resource planning.

## Dataset
Aadhaar Biometric Update dataset provided by UIDAI

## Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis
- Feature engineering
- Machine learning models including KNN and Random Forest

## Result
Random Forest regression achieved the best performance and identified district and state as key predictors.

## Tools Used
Python Pandas Matplotlib Scikit-learn
